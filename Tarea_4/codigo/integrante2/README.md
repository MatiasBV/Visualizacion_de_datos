## radar.py
Gráfico de **radar** que compara cuatro plataformas (Instagram, TikTok, YouTube, Facebook) en
cuatro ejes: usuarios (MAU), alcance publicitario, uso mensual entre usuarios 16+ y tiempo diario.
Los ejes se normalizan a 100 = máximo y se acompaña una tabla con los valores reales.
- Marca: línea + área (polígono).
- Canales: posición radial (valor normalizado por eje), color (plataforma), ángulo (variable).
- Fuente: Statista / We Are Social / DataReportal (oct. 2025). Datos definidos en el script.
- Salida: `viz_radar.png`.

Ejecutar desde esta carpeta: `python3 radar.py`
